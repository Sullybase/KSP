## SullyEngineer - [Full ReadMe](https://github.com/Sullybase/KSP/tree/main/SullyEngineer#readme)

I made this mod to have a better ease of playing on my low RAM computer. It has more RTGs, multi-resource storage tanks, and more. It is not on CKAN. I used the stock models/textures and base part configs then I edited the configs and used some AI.
